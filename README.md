# Family Dashboard

This repository contains a sample React + TypeScript family dashboard powered by Supabase.
The frontend code lives in the `frontend` directory.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in an `.env` file to connect to your Supabase project.
